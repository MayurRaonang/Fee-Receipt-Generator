# Fee Receipt Generator Website

A full-stack web application to manage student fee records and generate digital receipts for a coaching class.

## Features
- Admin login
- Dashboard with multiple class sections (Science, commerce, SSC)
- View and manage student fee records
- Fee payment form and PDF receipt generation
- Email/SMS alerts for payments

## Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express
- Database: MongoDB
